# Retail Sales Analysis Dashboard

## Project Overview
This project analyzes retail sales data to uncover insights about sales performance, profitability, and customer behavior.

## Dataset
Superstore Sales Dataset containing information about orders, sales, profit, categories, regions, and discounts.

## Tools Used
- Excel
- Power BI
- Power Query
- DAX

## Key Insights
- Identified top-performing product categories
- Analyzed regional sales performance
- Evaluated the impact of discounts on profit

## Dashboard
Interactive dashboard built using Power BI.
